# 게임 룰 명세 (원작 기준)

더 지니어스 시리즈에서 사용된 원작 룰을 나무위키(더 지니어스/역대 게임 및 개별 게임 문서)에서
확인해 정리한 문서. 구현 시 이 룰을 기준으로 하되, 각 게임의 "변형 방향" 항목대로 재설계한다.

> 표기: ✅ = 솔로(AI 대전) 적합 / 🌐 = 온라인 멀티 적합

---

## 1. 밀림장기 (원작: 십이장기 / 원류: 동물장기) ✅🌐

- 가로 4 × 세로 3, 총 12칸. 각자 앞쪽 3칸이 자기 진영.
- 기물 4종: **왕**(8방향 1칸), **장**(상하좌우 1칸), **상**(대각 4방향 1칸), **자**(앞 1칸, 상대 진영 진입 시 **후**로 승격 — 대각 뒤 2방향 제외 전방향).
- 잡은 기물은 포로가 되어 다음 턴부터 빈 칸에 드롭 가능(턴 소모). **상대 진영에는 드롭 불가**(동물장기와의 차이). 후를 잡으면 자로 되돌려 사용.
- 승리: 상대 왕을 잡거나, 자신의 왕이 상대 진영에 들어가 한 턴을 버티면 승리.
- 무승부 규칙 없음(원작). 원류인 동물장기는 후공 필승으로 완전 분석된 게임.
- **변형 방향(수축하는 밀림)**: N턴마다 가장자리 열 붕괴 → 교착 방지 + 미분석 게임화.

## 2. 콰트로 (원작: 콰트로 시즌4 ver. — 그랜드 파이널 9화) ✅🌐

시즌4 버전은 관전자 개입 없이 규칙 기반 "가상 플레이어"와 교환하는 순수 1:1 게임 (구현 채택 버전).

- 덱 26장: 빨강·파랑·노랑·초록 각 1~6 (24장) + 검정 0 카드 2장.
- **콰트로 = 숫자와 색상이 모두 다른 4장.** 0 카드도 별개 색·숫자로 인정.
- 각자 4장 배분, 마음에 안 들면 **멀리건 최대 2회**(새 4장으로 교환, 멀리건 완료 전까지 덱 재셔플 없음 → 본 카드 카운팅 가능).
- 남은 18장은 **가상 플레이어 6명에게 3장씩** 배분.
- 시작 시 각자 카드 1장 오픈. 오픈된 카드는 교환 불가.
- 교환 턴은 A-B-B-A-A-B… 2턴씩 교대. 자기 턴에 가상 플레이어 1명과 교환하거나 패스.
  - 내가 준 카드는 **전체 공개**, 가상 플레이어가 준 카드는 비공개(본인만 확인).
  - **가상 플레이어 교환 규칙**: ① 0 카드가 있으면 무조건 0을 준다 ② 교환자의 오픈 카드와
    양립하는(색·숫자 안 겹치는) 콰트로 최적 카드 ③ 없으면 최고 숫자 ④ 동수면 빨>파>노>초.
- **4장을 모두 오픈하기 전까지 가상 플레이어 6명 전원과 한 번씩 교환해야 한다.**
- 두 플레이어가 모두 패스하면 각자 다음 카드를 1장씩 오픈. 4장째 오픈 시 게임 종료.
- 승패: 둘 다 콰트로 → 합계 큰 쪽, 동점이면 최고 카드 큰 쪽 / 한쪽만 콰트로 → 콰트로 승 /
  둘 다 실패 → 합계 큰 쪽, 동점이면 최고 카드.
- **AI 포인트**: 공개 정보(오픈·건넨 카드·멀리건 카운팅)로 가상 플레이어 손패 추적,
  0 카드를 상대가 교환해야 할 가상 플레이어에게 주입하는 방해 전략.

## 3. 블라인드 포커 (원작: 인디언 포커) ✅🌐

- 1~10 카드 × 2장 = 20장. 각자 1장을 **자신만 못 보게** 이마에 부착.
- 기본 베팅 후 레이즈/콜/폴드. 콜이면 공개, 높은 숫자 승리.
- **10을 들고 폴드하면 페널티로 칩 10개** 지급. 무승부 팟은 다음 판 승자에게 이월.
- 20장 소진 시 새 덱 → 카드 카운팅 요소. 칩 전부 잃으면 패배.
- **AI 포인트**: 상대 폴드/레이즈 패턴의 베이지안 학습 + 카운팅.

## 4. 윷 대전 (원작: 전략 윷놀이) ✅(2인 변형)🌐

- 기본 룰은 일반 윷놀이와 동일 (잡기·업기·지름길·윷/모/잡기 한 번 더).
- 원작은 2:2 (탈락후보+파트너). 4명이 각자 **앞면 전용 윷과 뒷면 전용 윷 세트**를 받아
  하나씩 골라 던짐 → 윷 결과가 운이 아니라 4인의 동시 선택(심리전)으로 결정됨.
- **모든 도는 뒷도로 간주**되며, **뒷도로 출발점에 들어가도 1바퀴로 인정**된다.
- 각자 말 2개. 파트너 말 결과와 무관하게 탈락후보의 말 2개가 모두 들어와야 승리
  (파트너 말은 징검다리, 서로 잡을 수도 있음). 순서는 A-B-B-A.
- **변형(2인 룰)**: 각자 윷 2가락의 앞/뒤를 비밀 동시 선택, 4가락 합산(앞면 수 0=모,
  1=뒷도, 2=개, 3=걸, 4=윷) 결과로 현재 턴 플레이어의 말 이동. 심리전 코어 유지. 말 2개씩.

## 5. 윷과 거짓말 (원작: 의심 윷놀이 / 원류: 챠오챠오) ✅🌐(2인 대칭 변형)

- **윷판에서 일반 윷놀이 룰로 진행** (잡기·업기·지름길·한 번 더 전부 유지).
  윷가락 대신 **10면체 주사위**(도·개·걸 각 2면, 윷·모 각 1면, 꽝 2면) 사용. 뒷도 없음.
- 주사위 결과는 **굴린 본인만 확인** → 거짓 선언 가능. 이동할 말(과 경로)을 지정해 선언.
- **꽝**: 판 위 자기 말 1개 제거(본인 선택, 판에 말이 없으면 대기 말 제거). 이동이 없으므로
  의심 대상이 아님. 제거된 말은 재사용 불가 (잡힌 말은 제거가 아니라 출발 전으로 복귀).
- **의심**: 말을 이동시킬 때만 선언 가능. 적중(거짓) 시 **이동시켰던 말 제거**(업힌
  말은 2개 중 1개만 제거)·이동 무효. 실패(진실) 시 원작은 가넷 상실 —
  **2인 대칭 변형: 의심자의 말 1개 제거**(대기 우선), 이동은 그대로 진행.
- 윷/모는 즉시 이동 후 한 번 더(모아두기 불가). 잡으면 한 번 더 — 윷/모로 잡아도 추가는 1번만.
- **스타트 지점을 통과해야 완주 인정** (정확히 출발점에 서면 다음 이동으로 탈출).
- 원작 말 6개 중 2개 완주 시 우승 → 2인 변형도 말 6개·2개 선입선승, 남은 말이 2개 미만이면 패배.
- 원작의 관전 플레이어 가넷 베팅 요소는 2인 대칭 변형에서 제외.
- **AI 포인트**: 거짓말 빈도·타이밍을 학습하는 적응형 AI로 솔로 모드 성립.

## 6. 리플렉트 (원작: 레이저 장기 / 원류: Khet) ✅🌐

- 기물: **레이저**(이동 불가, 턴 종료 시 자동 발사, 파괴 불가), **왕**(어느 면이든 피격 시 즉시 패배), **세모기사** ×5(거울 1면 반사, 비거울면 피격 시 제거), **네모기사** ×2(동일 원리의 사각 기물), **스플리터**(레이저를 투과+반사로 2갈래 분리, 파괴 불가).
- 턴: 자기 기물 1개를 8방향 1칸 이동 **또는** 90° 회전(레이저는 회전만, 판 바깥 방향 불가). 턴 종료 시 자기 레이저 발사 → 비거울면 피격 기물은 아군 적군 불문 제거.
- **변형 방향**: 원작이 이미 Khet에서 변형된 구성. 기물 배치·수를 자체 밸런싱해 오리지널 초기 배치 제정.

## 7. 모노크롬 (원작: 흑과 백 / 모티브: 다빈치 코드) ✅🌐

- 각자 0~8 타일 9장. **0,2,4,6,8=흑 / 1,3,5,7=백** — 제시할 때 상대에게 색만 보임.
- 선 플레이어가 타일 1장을 엎어 제시 → 후 플레이어 제시 → 높은 숫자가 승점 1. **숫자는 승패 후에도 비공개**(색과 자기 기록만으로 추론).
- **순환 상성(변형 룰, 구룡투 모티브): 0은 8을 이긴다.** 그 외에는 큰 숫자가 승리.
  8이 무적이 아니게 되어 0이 버리는 패가 아니라 노림수가 되고, 8을 쥔 쪽도 0을 경계해야 한다.
- 승자가 다음 라운드 선. 9라운드 후 승점 多 승리, 동점 시 연장.
- **AI 포인트**: 제시 색 패턴 + 승패 정보로 상대 잔여 타일 베이지안 추적(승패 제약은
  숫자 대소가 아니라 0-8 순환을 포함한 실제 승부 결과 기준), 상대 성향 학습.

## 8. 모노크롬 II (원작: 흑과백 II / 모티브: 쿠오 바디스, 블로토 대령 게임) ✅🌐

- 원작 흑과백과 완전히 다른 게임. 각자 **99포인트**로 시작, 매 라운드 원하는 만큼 포인트 사용(1회성 소모).
- 제시 포인트가 **한 자릿수면 흑, 두 자릿수면 백**으로만 표시. 높은 쪽이 승점 1, 승자가 다음 선.
- 잔여 포인트는 5단계 게이지(0~19/20~39/40~59/60~79/80~99)로 공개.
- 9라운드 승점제, **5점 선취 즉시 승리**. 동점 시 33포인트로 3라운드 연장.

## 9. 야누스 포커 (원작: 양면포커) ✅🌐

- 앞뒷면에 서로 다른 숫자가 적힌 양면 카드. 그랜드 파이널 기준: 1~10, **앞뒤 홀짝이 반드시
  다름**(앞이 짝수면 뒤는 홀수) — 50종 × 4세트 = 200장. 칩 40개씩, 기본 베팅 1개씩.
- 카드 1장씩 배분: **앞면은 서로 공개, 뒷면은 본인만 확인**.
- 선부터 자기 카드의 앞/뒷면 중 **베팅할 면을 공개 선언**하고 베팅(이후 변경 불가).
  더 많이 걸면 베팅 지속(레이즈), 같은 개수를 걸면(콜) 즉시 종료 → 선택 면 공개, 높은 숫자 승.
- **선은 베팅 없이 포기 가능**(상대의 양면베팅 기회 봉쇄 전략). 포기하면 상대 승리이며
  **포기한 카드의 뒷면은 공개되지 않는다**(카운팅 불가). 무승부 팟은 다음 판 승자에게 이월.
- **양면베팅**: 자기 양면이 모두 상대 선택 면을 이길 것 같으면 선언 — 앞·뒤 두 곳에 각각
  같은 금액 베팅(항상 2배 지불). 양면 모두 높아야 승리(+상대 칩 10개 추가 획득), 한 면이라도
  같거나 낮으면 패배. 상대는 양면베팅에 폴드해도 칩 10개를 내야 하지만,
  **양면베팅한 쪽은 패배해도 10개를 내지 않는다** (선공 양면베팅이 저위험).
- 칩을 모두 잃으면 패배.
- **AI 포인트**: 공개 정보 3/4 구조라 확률 계산 + 홀짝 추론(짝수 앞면 → 뒷면 최대 9) +
  공개된 카드 카운팅(폴드된 뒷면은 알 수 없음) + 상대 면 선택/폴드 성향 학습.

## 10. 암전 미궁 (원작: 마법의 미로 Das magische Labyrinth) ✅(공개 정보 기억 AI)🌐

- **6×6 판**, 칸 사이에 **보이지 않는 벽 24개**(원작 어려움 기준; 쉬움 19개). 모든 칸은 항상
  도달 가능하게 배치(스패닝 트리 보장 + 루프 1개). 2인전은 **대각 맞은편 코너**에서 출발.
- 매 라운드 **목표 심볼 칩을 공개로 추첨** — 해당 심볼 칸에 먼저 도달하는 쪽이 칩 획득.
  이미 그 칸에 서 있으면 즉시 획득하고 새 칩 추첨.
- 자기 턴: **주사위(눈 1,2,2,3,3,4)** 를 굴려 나온 수 **이하**만큼 상하좌우로 한 칸씩 이동
  (남은 걸음 포기 가능, 대각선 불가, 상대 말 통과 가능·같은 칸 공존 가능).
- 보이지 않는 벽을 지나려 하면 **쇠구슬이 떨어지며 남은 걸음 몰수, 자기 시작 코너로 복귀**
  (부딪힌 위치는 양쪽 모두에게 공개되는 정보 — 기억하는 자가 이긴다).
- 목표 칸에 도달하면 칩을 얻고 그 턴 종료. **칩 5개를 먼저 모으면 승리**.
- **AI 포인트**: AI는 벽 배치를 볼 수 없고, 양쪽이 부딪히거나(벽 확정) 통과한(개방 확정)
  간선 등 공개 정보만 완벽 기억. 미지 간선은 잔여 벽 수 기반 확률로 위험도를 계산해
  기대 비용 최단 경로를 탐색. 인간의 기억력 vs AI의 완벽 기억이 승부처.

## 11. 순환선 (원작: 모노레일 / 원류: 베니스 커넥션) ✅🌐

- 시작: 철로가 양끝으로 열린 **기차역 타일 2개가 가로로 나란히**(─ ─) 놓인 상태. 철로 타일 16개(양면: 직선/ㄱ자 — 어느 면이든 선택 가능).
- 번갈아 **1~3개**를 놓되 2개 이상이면 **나란히 일렬로**, 각 타일은 기존에 놓인 타일의
  상하좌우 중 한 면에 맞닿아야 함(철로가 이어지게 놓을 필요는 없음). 시간 제한 없음.
- 역에서 출발해 역으로 돌아오는 **하나의 순환선**을 완성하는 마지막 타일을 놓는 쪽이 승리.
  16개를 모두 쓸 필요는 없으나 **이미 놓인 타일은 전부 순환선에 포함**되어야 한다
  (기차역은 가로 직선이므로 순환선이 역을 좌우로 통과).
- **불가능 선언**: 자기 턴 시작 시 "남은 타일로 완성 불가"를 선언 가능 → 상대가 남은
  타일로 완성하면 상대 승, 실패하면 선언자 승.
- 선공 필승이 알려진 게임(첫수 150가지 중 5가지 필승, 나무위키 모노레일(게임) 문서에
  주형별 필승 여부 정리) → **AI 포인트**: 실행 불가능 상태 즉시 판정 + 메모이제이션
  완전 탐색으로 신급 AI.

## 12. 모노크롬 레이즈 (원작: 베팅! 흑과 백) ✅🌐

- 각자 숫자 타일 10개(0~9: 짝수 흑, 홀수 백) + 칩 30개.
- 시작 전 가림막 상태에서 **타일 10개의 대결 순서를 배치**하고 **칩 30개를 각 타일에 분배**(타일당 최소 1개, 전부 베팅). 이후 가림막 제거, 1번 타일부터 순서대로 대결.
- 각 대결에서 칩을 적게 건 쪽이 선택: **콜**(칩 개수 맞추기 — 부족하면 다른 타일에서 차출 가능, 단 각 타일 최소 1개 유지) 또는 **폴드**(타일 비공개, 베팅 칩은 상대에게).
- 콜이면 공개, 높은 숫자가 팟 획득(무승부는 각자 회수). 10라운드 후 칩 많은 쪽 승리(총 60개 중 31개 이상).
- **AI 포인트**: 배치·분배가 승부의 대부분 — 블로토 게임식 전략 생성 + 상대 과거 배치 학습.

## 13. 수(數)의 진 (원작: 숫자장기 / 모티브: 스트라테고) ✅🌐

- 6열 × 9행 = 54칸(각자 앞 3행이 자기 진영). 기물: 숫자 1~10 각 1개 + **지뢰 3**(이동 불가) + **왕 1** = 14개. **뒷면 배치(비공개)**.
- 이동: 좌우·대각 1칸, 앞으로 최대 2칸(점프 불가), 뒤로 불가. 이동 후 상하좌우로 적 기물과 맞닿으면 즉시 대결(다중 대결은 동시 판정).
- 대결: 두 숫자의 **합이 10 이상이면 큰 수 승리, 10 미만이면 작은 수 승리**. 판 위 18개의 **마이너스 표식**을 사이에 두면 차(큰수-작은수) 대결. 동수는 상쇄. 지뢰는 자폭(둘 다 제거). 왕은 대결 시 즉사(왕끼리는 무효).
- 상대 진영 끝줄 도달 시: 그 기물을 제거하고 죽은 기물 1개를 공개 상태로 부활(자기 끝줄 배치).
- 승리 4조건: 시간 초과 / 왕 포획 / 왕 제외 전멸 / **자신의 왕이 상대 끝줄 도달**.
- 원작 아이템(+1, -1, BLIND)은 선택 요소로.

## 14. 히든 포뮬러 (원작: 미스터리 사인 / 모티브: 파라오코드) ✅🌐

- 1:1 버저 게임, 총 **11라운드**. 라운드마다 `X ? Y` 형태의 문제 공개 (?는 숨겨진 연산 규칙).
- 두 플레이어가 **0을 제외한 수를 하나씩 제시**(선이 앞, 후가 뒤에 대입) → 숨은 규칙에
  대입한 `A ? B = C`가 **힌트**로 공개. 문제의 두 수를 하나만 쓰는 건 가능하나 **둘 다
  동시에 쓰는 것은 금지**.
- 힌트 공개 후 **1분 내 버저** → 문제의 정답 제시. 정답 시 **+1**, 오답 시 **−1**이며
  기회가 상대에게 넘어간다. **버저를 누른 뒤에는 10초 안에 답해야 하며, 넘기면 오답 처리**
  (원작의 긴장감 재현 — 확신 없이 선점하려 버저를 누르면 손해). 아무도 못 맞히면 다시 수 제시 → 힌트 추가(**최대 16개**,
  이후 다음 문제로). 11라운드 후 승점 높은 쪽 승리(동점 시 연장 라운드).
- 원작 출제 규칙(그랜드 파이널 결승): 큰수+차 / 합·차 이어쓰기 / 큰수÷작은수의 나머지 /
  앞뒤 자릿수합의 곱 / 이어쓴 수의 연속 숫자 개수 표기 / 모든 자리 숫자 합 / (합÷12)의
  나머지(시계) / 동그라미 개수 / 합×뒤의 수 — 계산·비계산 규칙 혼합.
- 변형: 원작의 결승 전용 아이템(DOUBLE/SECRET/무감점)은 제외. 원작은 정답 풀이를
  상대에게 비공개하지만 본작은 **라운드 종료 후 규칙 공개**(솔로 UX·AI 투명성).
- **AI 포인트**: AI는 숨은 규칙을 읽지 않고 **규칙 은행 후보를 힌트와 대조해 소거**
  (베이지안 필터링). 남은 후보가 하나로 확정되거나 모든 후보의 답이 일치할 때만 버저 —
  따라서 오답이 없다. 수 제시는 후보 판별력을 최대화하는 능동 실험. 난이도는 라운드별
  고려 규칙 폭으로 조절하며 인간의 정답률·버저 속도를 학습해 보정.
